# راهنمای به‌روزرسانی قیمت‌ها از سایت اصلی

## روش 1: استفاده از پنل ادمین (توصیه می‌شود)

1. وارد پنل ادمین شوید (سه بار روی لوگو کلیک کنید)
2. وارد تب "محصولات" شوید
3. برای هر محصول:
   - روی دکمه "ویرایش" کلیک کنید
   - قیمت را از سایت https://riracafe.menusaz.com کپی کنید
   - در فیلد قیمت وارد کنید
   - ذخیره کنید

## روش 2: ویرایش مستقیم فایل

فایل `/data/menuData.ts` را باز کنید و قیمت‌ها را به‌روزرسانی کنید:

```typescript
{
  id: 1,
  name: "Espresso Single",
  nameFa: "اسپرسو سینگل",
  price: "38,000", // قیمت را اینجا تغییر دهید
  ...
}
```

## مراحل بررسی و به‌روزرسانی

1. به سایت https://riracafe.menusaz.com بروید
2. منوی کامل را باز کنید
3. قیمت هر آیتم را یادداشت کنید
4. با لیست بالا مقایسه کنید
5. تفاوت‌ها را در پنل ادمین یا فایل به‌روزرسانی کنید

## نکات مهم

- قیمت‌ها را با کاما وارد کنید (مثال: 45,000)
- اگر محصولی در سایت اصلی نیست، می‌توانید آن را حذف کنید
- اگر محصول جدیدی در سایت اصلی است، آن را اضافه کنید
- دقت کنید نام فارسی و انگلیسی محصولات مطابقت داشته باشد

## مثال عملی

اگر در سایت اصلی قیمت "اسپرسو سینگل" 40,000 تومان است:

1. در پنل ادمین محصول را پیدا کنید
2. روی ویرایش کلیک کنید
3. قیمت را از 38,000 به 40,000 تغییر دهید
4. ذخیره کنید

یا در فایل:
```typescript
{
  id: 1,
  name: "Espresso Single",
  nameFa: "اسپرسو سینگل",
  price: "40,000", // تغییر از 38,000 به 40,000
}
```
